À propos de ce projet

Le but de ce projet était de me familiariser avec le framework Angular, qui est utilisé pour développer des applications web en une seule page, également appelées "single page app".

L'objectif est d'améliorer l'expérience utilisateur en optimisant le temps de chargement, car tous les composants sont chargés sur une même page.

Pour réaliser ce projet, j'ai dû me familiariser avec TypeScript, qui est le langage de programmation utilisé par Angular.

En outre, j'ai créé une API pour une base de données MySQL et l'ai intégrée à mon application.

La page de gestion permet de réaliser toutes les opérations nécessaires pour communiquer avec la base de données, comme POST, PUT, ADD et DELETE.
